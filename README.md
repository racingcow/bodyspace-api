# BodySpace API

BodySpace has no public API. Let's fix that.

## Install and Run

Install .NET Core 1.1

`https://www.microsoft.com/net/core`

Clone

`git clone https://github.com/racingcow/bodyspace-api.git`

Restore Dependencies

`dotnet restore`

Build

`dotnet build`

Run

`dotnet run`

Open Web Brower

`http://localhost:5000/api/ping`
